## About This Project

AI_News is a Python CLI tool that aggregates AI/ML news from multiple sources (RSS, Hacker News, Reddit, arXiv),
scores and sorts articles by relevance and recency, and generates HTML digest pages hosted on GitHub Pages.
The goal is a daily curated AI news briefing with minimal manual effort.
